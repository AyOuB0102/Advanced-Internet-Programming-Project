منظومة ويب متكاملة لإدارة العمل البحثي والمشاريع الأكاديمية، صُمّمت خصيصًا لتكون Front-End فقط باستخدام تقنيات الويب الأساسية، مع دعم العمل دون اتصال (Offline-First)
فكرة المشروع
يهدف ResearchHub إلى مساعدة الطلبة والباحثين على:
- تنظيم المشاريع البحثية
- إدارة المهام باستخدام لوحة Kanban
- حفظ وتتبع المراجع العلمية (Papers)
- تحليل التقدم عبر رسوم بيانية
- العمل بدون خادم (Serverless) وبلا قواعد بيانات

- التقنيات المستخدمة
- HTML5
- CSS3 (Responsive + Dark / Light Mode)
- JavaScript (Vanilla JS)
- localStorage لحفظ البيانات محليًا
- Canvas API لرسم الرسوم البيانية
